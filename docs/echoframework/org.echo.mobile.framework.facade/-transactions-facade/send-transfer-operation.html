sendTransferOperation

abstract fun sendTransferOperation(nameOrId: String, wif: String, toNameOrId: String, amount: String, asset: String, feeAsset: String?, callback: Callback<Boolean>): Unit

Submits transaction to blockchain

Parameters

nameOrId - Source account name or id

wif - Account's private key in wif format

toNameOrId - Transfer target account name or id

amount - Value amount of transfer operation

asset - Specific asset type id

feeAsset - Specific asset type id for calculating fee. If null - asset will be used

callback - Listener of operation results. Retrieves true if transfer succeed, otherwise - false
