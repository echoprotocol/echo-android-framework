AccountBalancesSocketOperation

class AccountBalancesSocketOperation : SocketOperation<AssetAmount>

This function fetches all relevant Account objects for the given accounts, and subscribes to updates to the given accounts. If any of the strings in nameOrId cannot be tied to an account, that input will be ignored. All other accounts will be retrieved and subscribed.

asset - Required asset type
