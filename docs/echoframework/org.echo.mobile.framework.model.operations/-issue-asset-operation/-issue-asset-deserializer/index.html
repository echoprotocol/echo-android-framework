IssueAssetDeserializer

class IssueAssetDeserializer : JsonDeserializer<IssueAssetOperation>

This deserializer will work on any transfer operation serialized in the 'array form' used a lot in the Graphene Blockchain API.

An example of this serialized form is the following:

{ "fee": { "amount": 264174, "asset_id": "1.3.0" }, "from": "1.2.138632", "to": "1.2.129848", "amount": { "amount": 100, "asset_id": "1.3.0" }, "extensions": [] }

It will convert this data into a nice TransferOperation object.
