fun Int.revert(): ByteArray

Returns an array of bytes with the underlying data used to represent an integer in the reverse form. This is useful for endianess switches, meaning that if you give this function a big-endian integer it will return it's little-endian bytes.

Parameters

input - An Integer value.

Return
The array of bytes that represent this value in the reverse format.
